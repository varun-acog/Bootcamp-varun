Research Papers Explorer - Notes

Approaches

  • Dynamic Filtering: Implemented multiple filter options (author, title, year range) using JavaScript functions to dynamically filter research papers based on user input.
  • Search Functionality: Added a general search input that performs case-insensitive searches across all fields of the research paper dataset.
  • Dynamic Data Rendering: Used JavaScript to dynamically create and render research paper cards from the JSON dataset.
  • Read More Feature: Designed a toggle feature for the abstract field to show truncated content with a "Read more" link, expanding to the full text when clicked.
  • Dark Theme and Styling: Adopted a modern dark theme with a focus on accessibility and aesthetics. Used consistent spacing, borders, and typography to enhance readability.
  • Export Filtered Data: Provided functionality to export filtered research paper data into a downloadable JSON file.

Challenges

  • Author Filter Bug: Initially, the author filter was not functioning because the incorrect field (`paper.author`) was referenced instead of `paper.authors`. This was fixed by updating the field reference.
  • Efficient Filtering: Combining multiple filters (author, title, year range) posed a challenge. It was resolved by chaining logical conditions within the filter function.
  • Abstract Handling: Displaying long abstracts concisely required implementing a "Read more" toggle while ensuring a smooth user experience. Adding event listeners dynamically to each card was initially problematic but resolved using inline `onclick` attributes.
  • Styling Consistency: Balancing aesthetic design with readability in a dark theme required iterative testing of color schemes, font sizes, and spacing.
