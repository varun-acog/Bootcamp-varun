Research Paper Explorer (Using React)

Overview

A Research Paper Explorer application built using React and Vite with vanilla CSS styling. The app allows users to search, filter, and export research papers based on various criteria.

Features

  • Search research papers by any field.
  • Filter by author, title, and year range.
  • Export filtered papers as a JSON file.
  • Read more/less functionality for paper abstracts.

File Structure

  src/
  ├── App.jsx               # Main application component
  ├── components/
  │   ├── Header.jsx        # Header component with search functionality
  │   ├── Filters.jsx       # Filters for author, title, and year
  │   ├── PaperCard.jsx     # Individual card for each research paper
  ├── styles/
  │   └── styles.css        # Vanilla CSS for styling
  ├── assets/
  │   └── data.json         # Research papers data
  └── main.jsx              # Entry point for the React app

Key Components

App.jsx

  • Manages state for papers and filtered results.
  • Handles search, filters, and data export functionality.

Header.jsx

  • Renders the application title and search bar.
  • Passes search query to the parent component.

Filters.jsx

  • Provides inputs for filtering by author, title, and year.
  • Triggers filtering logic in the parent component.
  • Includes an export button for filtered data.

PaperCard.jsx

  • Displays paper details like title, author, venue, year, ID, and abstract.
  • Implements read more/less toggle for abstracts.

Styling

Vanilla CSS for dark-themed styling with a responsive layout:

  • Modern design with flexbox for layout.
  • Dark background, light text, and contrasting highlights.
  • Interactive elements with hover effects.

Setup and Run

  1. Create a Vite React project: npm create vite@latest research-paper-explorer --template react
  2. Install dependencies: npm install
  3. Place files as per the structure above.
  4. Run the app: npm run dev

Data Format (data.json)

  [
    {
      "id": "1",
      "title": "Sample Paper",
      "authors": "Author Name",
      "venue": "Conference Name",
      "year": 2023,
      "abstract": "This is a sample abstract for the research paper."
    },
    ...
  ]

Export Functionality

Generates a downloadable JSON file containing filtered research paper data.

Run Command

To start the development server:

npm run dev

Summary

The Research Paper Explorer demonstrates efficient management of data and UI using React and Vite. Its modular components make it maintainable, and its functionality is user-focused.
