ReactJS Weather App Notes

Project Overview

A weather app built with ReactJS, styled using Tailwind CSS, and enhanced with ShadCN UI components.

Steps to Build

  1. Set Up React App: Create a React app using npx create-react-app weather-app.
  2. Install Tailwind CSS:
    • Install Tailwind: npm install tailwindcss postcss autoprefixer
    • Initialize: npx tailwindcss init
    • Add Tailwind directives to src/index.css: 
      @tailwind base;
@tailwind components;
@tailwind utilities;
  3. Add ShadCN Components: Install ShadCN: npm install @shadcn/ui.
  4. Update Tailwind Config: Modify tailwind.config.js: 
    module.exports = {
  content: ["./src/**/*.{html,js,jsx,ts,tsx}"],
  theme: { extend: {} },
  plugins: [],
};
  5. Create Components:
    • App.js: Manages city input and renders CityWeather. Uses ShadCN's Button and Input.
    • CityWeather.js: Displays the city name and temperature. Simulates fetching weather data with useEffect.

Running the App

Start the app using npm start.

Key Features

  • Real-time weather data simulation using useState and useEffect.
  • Tailwind CSS for modern and responsive styling.
  • ShadCN UI components for enhanced user experience.
