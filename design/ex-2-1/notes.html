Font Exercises Notes

Level 1: Basics of Fonts

Key Learnings:

  • Understanding system fonts such as Arial, Times New Roman, and Courier New.
  • Adding a small JavaScript snippet to enhance interactivity, like dynamically changing styles.

Challenges Faced:

  • Keeping the JS snippet simple yet meaningful within the constraints.

Level 2: Using Google Fonts and Advanced Text Styling

Key Learnings:

  • How to include Google Fonts using the `` tag.
  • Creating text effects such as:
    • Drop cap using `::first-letter`.
    • Text shadow with `text-shadow` property.
    • Image-based text using `-webkit-background-clip`.

Challenges Faced:

  • Fine-tuning effects like shadows and background-clip text for readability.

Level 3: Custom Fonts and Typographic Hierarchy

Key Learnings:

  • Downloading and installing custom fonts locally using `@font-face`.
  • Combining multiple font families (e.g., Playfair Display for headers and Geist for body) to create a typographic hierarchy.

Challenges Faced:

  • Ensuring custom fonts load correctly across browsers (especially offline fonts).
