Color Exercises Notes

Level 1: Basics of Colors

Key Learnings:

  • Understanding color theory and creating color schemes like monochromatic and complementary.
  • Using CSS color formats: Hex, RGB, HSL, and alpha channels for transparency.
  • Creating a three-color scheme by incorporating an accent color to differentiate elements like headers, links, and buttons.

Challenges Faced:

  • Balancing the visual contrast between elements in the complementary color scheme to ensure clarity.

Level 2: Dark and Light Themes

Key Learnings:

  • Creating two separate stylesheets for light and dark themes and applying them using media queries or a toggle.
  • Styling images with overlaid text to ensure that text is readable in both themes with proper contrast adjustments.
  • Building a color swatch page using HSL values to generate variations in saturation and lightness.

Challenges Faced:

  • Finding the right level of saturation and lightness for HSL swatches to make them visually distinguishable yet harmonious.

Level 3: Advanced Color Design

Key Learnings:

  • Designing gradient themes using CSS (linear gradients for headers and radial gradients for buttons).
  • Creating a responsive color palette with tints and shades by using `lighten()` and `darken()` functions in CSS for better visual impact across different screen sizes.
  • Using borders and shadows to enhance visual hierarchy and clearly differentiate sections in the design.

Challenges Faced:

  • Making the gradient transitions smooth and visually pleasant, especially when working with multiple gradient stops.
  • Ensuring that the responsive color palette adapted correctly to various screen sizes without breaking the design.
