Icons Exercise Notes

Level 1: Basics of Icons

Approach:

  • Used Font Awesome as the icon font library.
  • Added three icons to the page (e.g., home, user, and settings).
  • Styled the icons using CSS for size, color, and spacing.

Key Learnings:

  • Icon font libraries like Font Awesome simplify the process of adding and customizing icons.
  • CSS properties like `font-size`, `color`, and `margin` make it easy to style icons.

Challenges Faced:

  • Ensuring consistent scaling of icons across different screen sizes.
  • Choosing the right icons to convey meaning effectively.

Level 2: Using SVG Icons

Approach:

  • Replaced one Font Awesome icon with an inline SVG file.
  • Created a grid of six icons: three from Font Awesome and three as inline SVGs.
  • Added hover effects such as color changes and scaling using CSS transitions.

Key Learnings:

  • SVGs offer more flexibility for customization compared to icon fonts, such as finer control over colors and shapes.
  • Hover effects enhance user experience and provide interactive feedback.
  • CSS transitions can be used to add smooth effects to icons for better aesthetics.

Challenges Faced:

  • Managing alignment between font-based icons and SVG icons in the grid.
  • Ensuring hover effects were subtle yet noticeable.

Level 3: Navigation Bar and Icon Buttons

Approach:

  • Designed a navigation bar with icons for home, about, and contact sections, aligned with text labels using CSS flexbox.
  • Created download and upload buttons with embedded SVG icons.
  • Applied consistent spacing, alignment, and hover effects to all icons and buttons.

Key Learnings:

  • Icons in navigation bars improve clarity and usability, especially for mobile users.
  • SVGs can be embedded directly into HTML for highly customizable icons in buttons.
  • Using CSS flexbox simplifies alignment of icons and text in navigation bars and buttons.

Challenges Faced:

  • Maintaining consistent spacing and alignment for icons and labels across different screen sizes.
  • Ensuring all icons had uniform visual weight and size, especially when mixing font icons and SVGs.
  • Creating hover effects that worked equally well for navigation icons and buttons.
