Layout Principles Exercise Notes

Level 1: Recreating a Simple Webpage Layout

Approach:

  • Chose a simple article page layout inspired by medium.com.
  • Applied consistent margins and padding using CSS to ensure proper spacing between elements.
  • Used white space strategically to separate sections and improve readability.
  • Aligned text and images with CSS properties like `text-align`, `margin`, and `padding`.

Key Learnings:

  • The importance of white space in creating visually appealing and readable layouts.
  • Text and image alignment are critical for maintaining a polished and professional look.

Challenges Faced:

  • Ensuring consistent spacing across different screen sizes.
  • Choosing appropriate padding values without overcrowding the content.
