Website Review Notes

Website Reviewed: India.gov.in News Lists

1. Problems Identified

Below are the problems annotated using the Awesome Screenshot tool:

  • Lack of Visual Hierarchy: The text-heavy layout makes it hard to identify important news items. Font size and weight are similar across the page, reducing clarity.
  • Poor Navigation: The news list lacks categorization and filtering options, making it overwhelming for users to find relevant updates.
  • Outdated Design: The website uses an older HTML design with minimal spacing, cluttered content, and lacks modern styling.
  • No Responsiveness: The page does not render well on mobile devices, which impacts usability on smaller screens.
  • No Hover effects: There are no hover effects for buttons which makes them difficult to find whether they are clickable or not.
  • Overloaded Links: Too many links placed close together make navigation confusing and reduce readability.

3. Suggested Improvements

Based on the issues identified and inspirations, here are my recommendations to improve the website:

  • Improve Visual Hierarchy: Use larger, bold fonts for headings, increase spacing, and highlight key news items.
  • Add Filtering Options: Allow users to sort news by date, category, or relevance.
  • Modernize Design: Use a responsive layout with CSS Grid or Flexbox to organize content. Add visuals like thumbnails or icons to break up text.
  • Enhance Mobile Responsiveness: Ensure the site scales well across devices by implementing a responsive design.
  • Reduce Link Overload: Group related links into dropdowns or sections to reduce clutter.
  • Add Hover effects: Should add hover effects for buttons which makes them easy find whether they are clickable or not.
