Website Analysis: CSE IITM Faculty List

Annotated Issues

Here are the problems identified on the CSE IITM Faculty Page using the Awesome Screenshot tool:

  • Lack of Visual Hierarchy:

    The page presents faculty details in a plain text format without headings, proper spacing, or bold highlights.

  • Poor User Interface:

    The design is outdated, with no modern styling, color palette, or responsiveness for mobile screens.

  • Unclear Navigation:

    There are no filters, categories, or sorting options to locate faculty members easily (e.g., by research area or department).

  • Alignment Issues:

    Faculty names, designations, and email IDs are not aligned properly, leading to poor readability.

  • Email formatting:

    Email id formatting is very improper and unreadable.

Recommendations for Improvement

  • Add Visual Hierarchy:

    Use bold titles, headings, and proper spacing to separate faculty names, designations, and contact details.

  • Implement Filters and Search:

    Allow users to search for faculty members based on their research area, alphabetical order, or department.

  • Use CSS Grid for Layout:

    Organize content using a grid-based layout for a clean and modern structure.

  • Update the Design:

    Use a modern font, color palette, and spacing to improve visual clarity and aesthetics.
